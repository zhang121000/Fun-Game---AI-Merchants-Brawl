# AI Health Mall · Fun Game · AI Merchants Brawl

[English](README.md) | 简体中文

AI 健康生活馆是一个保健品电商竞争模拟平台。5 个 AI 商家分别经营不同品类的保健品，平台会根据每日经营表现动态分配顾客流量，商家 AI 再独立决定定价、促销、人群定位和补货策略，最终通过概率引擎生成订单并更新排名。

> 灵感来源：抖音创作者 **梯度下沉站**

---

## 项目亮点

- 5 个独立 AI 商家，分别接入不同的 OpenAI 兼容模型
- 平台 AI 每日按经营表现分配 500 个模拟顾客流量
- 商家 AI 独立决策定价、促销、目标人群、卖点和补货
- 概率引擎结合折扣、促销、品类匹配等因素生成订单
- 支持“推进明天”、排行榜、经营分析、营销策略和后台管理

## 顾客池

系统内置一个 **5000+ 模拟顾客池**，包含 18 位明星用户和大量按人口比例生成的统计用户。

每位顾客都拥有独立属性：

- 品类偏好向量
- 价格敏感度
- 品牌忠诚度
- 时间行为模式

## 每日模拟流程

点击“推进明天”或调用 `POST /api/v1/admin/advance-day`，即可触发完整的一日经营循环。

### 1. 平台 AI 分配流量

平台调度 AI 会读取昨天的销售、收入、排名和趋势数据，再结合 4 类人群比例，将 500 个顾客流量动态分配给 5 个商家。

分配原则包括：

- 正向激励：卖得好的商家获得更多流量
- 底线保护：垫底商家仍保留最低流量，保证翻盘机会
- 品类匹配：如钙片更偏向老年人，蛋白粉更偏向青年人
- 探索机制：偶尔给强者分配弱匹配人群，测试边界

如果平台 AI 调用失败，系统会自动降级为基于品类-人群匹配度的加权分配，保证模拟不中断。

### 2. 5 个商家 AI 并行决策

5 个商家 AI 同时收到经营报告，并在 15 秒超时内完成决策。每个 AI 需要决定：

| 决策项 | 说明 |
|---|---|
| 定价 | 在当前价格基础上做有限幅度调整 |
| 促销 | 限时折扣、满减、买赠等 |
| 目标人群 | 今日重点投放的人群 |
| 卖点文案 | 优化产品描述与推荐话术 |
| 补货 | 库存不足时自动补货 |
| 新品研发 | 垫底时可启动 3 天游研流程 |

AI 提示词会包含昨天销量、昨天收入、7 日均值、排名趋势、竞品排名，以及平台当日预计分配流量。

### 3. 概率引擎生成订单

每一笔潜在交易都会经过多层概率计算：

```text
最终转化率 = 基础转化率 × 价格折扣因子 × 促销因子 × 品类匹配度
```

- 价格折扣因子：`1 + 折扣率 × 2`
- 促销因子：有促销时乘以 `1.3`
- 品类匹配度：例如钙片对老年人更高，蛋白粉对儿童更低
- 转化率上限：最高 60%，防止极端策略失衡
- 随机扰动：每笔交易独立抽样，模拟真实市场噪声

### 4. 排名更新与平台建议

每天结束后，系统按销量重新排序，并自动检测异常：

- 连续垫底：触发警告，建议降价或研发新品
- 有流量无转化：建议检查定价和卖点文案

## 新品研发系统

垫底商家可以启动 **新品研发**，进入 3 天游研周期。研发完成后会在原品类下推出一个差异化新品，并自动参与竞争。

研发进度会出现在每日决策日志中，便于管理员追踪。

---

## 管理后台

管理员通过 `/admin` 控制整个模拟：

- 推进明天：一键触发完整的一日循环
- 实时排行榜：查看销量、收入、定价、促销、人群等信息
- 决策链路：查看每个 AI 的完整决策理由
- 平台建议：自动识别垫底和零销量，给出策略提示
- 研发追踪：查看所有进行中和已完成研发项目
- 库存管理：支持批量或单个补货
- 一键重置：清空模拟数据并从第 0 天重新开始

## 快速开始

```bash
# 1. 后端（端口 8000）
cd backend
pip install -r requirements.txt
cp .env.example .env
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 2. 前端（端口 5173，/api 代理到 8000）
cd frontend
npm install
npm run dev
```

浏览器打开 `http://localhost:5173`，进入管理后台后点击“推进明天”即可开始模拟。

## API 说明

常用管理接口：

- `POST /api/v1/admin/advance-day`：推进一天
- `GET /api/v1/admin/advance-day-status`：查询推进进度
- `POST /api/v1/admin/generate-strategies`：生成营销策略
- `POST /api/v1/admin/reset`：重置模拟
